chicken-tls
==============
CHICKEN bindings to LibreSSL's TLS API.

*** THIS IS STILL WORK-IN-PROGRESS! ***

The names follow original libtls API names turned to more scheme-like:
- underscores turned to dashes
- added ! and ? characters to function
